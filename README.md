# Success Popup UI Component
#### Интерактивный UI-компонент модального окна подтверждения успеха. Проект выполнен с акцентом на плавность пользовательского опыта (UX), чистоту кода и современную визуальную эстетику.

# Стек технологий
#### HTML5: Семантическая структура документа.
#### CSS3: * Flexbox для идеального центрирования элементов.
#### CSS Animations для улучшения визуального отклика.
#### Работа с прозрачностью и позиционированием (fixed, inset).
#### JavaScript (Vanilla JS): Лаконичные функции для манипуляции CSS-свойствами элементов в реальном времени.
#### Font Awesome: Интеграция векторных иконок для улучшения визуальной информативности.

# Основной функционал
#### Плавная анимация: Использование @keyframes ScaleIn для мягкого появления карточки.
#### Интерактивность: Управление состоянием окна (открытие/закрытие) реализовано на чистом JavaScript без использования тяжелых библиотек.
#### Современный UI: Реализация стилей в духе минимализма с использованием градиентов, скругленных углов и продуманной типографики.
#### Overlay эффект: Затемнение заднего фона для фокусировки внимания пользователя на важном уведомлении.

# Структура проекта
#### index.html — Разметка структуры окна и кнопки вызова.
#### cstyle.css — Стилизация интерфейса, анимации и адаптивные параметры.
#### script.js — Логика открытия и закрытия модального окна.
